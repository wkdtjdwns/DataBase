# 자료 (Data)와 정보 (Information)의 차이

- 자료 : 3 / 8 맑고 추움
- 정보 : 3 / 8 맑고 춥기 때문에 따듯하게 입어야 함.
- 위와 같이 자료는 그저 데이터지만 정보는 그 자료를 가공한 것임.

<br>

# **자료의 구성 단위의 크기**

Bit(비트) - Nibble(니블) - Byte(바이트) - 워드(Word) - 필드(field) - 레코드(Record) - 파일(File) - 데이터베이스
